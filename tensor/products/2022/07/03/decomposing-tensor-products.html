Decomposing Tensor Products

Jul 3, 2022

Angular Momentum

A classic example of where Clebsch-Gordan coefficients arise in physics is the decomposition of combined spin systems into direct sums of angular momentum representations.

The Pauli Matrices

For a single spin-\(\dfrac{1}{2}\) particle, the angular momentum operators in the \(x, y, z\) directions respectively are represented by the Pauli matrices, as follows:

\[J_x = \dfrac{1}{2}\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}\] \[J_y = \dfrac{1}{2}\begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}\] \[J_z = \dfrac{1}{2}\begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}\]

\(J^2 = J_x^2 + J_y^2 + J_z^2\) \(J_{\pm} = J_x \pm iJ_y\)

The Pauli matrices satisfy the commutation relations: \([J_x,J_y] = 2iJ_z\) (and cyclic permutations of the labels \(x, y, z\)). \([J_z, J_{\pm}] = \pm J_{\pm}\) \([J^2, J_z] = 0\)

\[J_x + iJ_y = \dfrac{1}{2}\begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\]

2 Spin Systems

For the particular case of combining 2 spin systems, we can explicitly write out the operators involved. Working in the z-basis, we may write:

\[J_{z,total} = JzA \otimes I + I \otimes JzB\]

In a similar manner, we can obtain all of the necessary angular momentum operators for such a 2-state system.

For spin-\(\dfrac{1}{2}\) we have \(\vert j = 1/2, m\rangle\) and the relation \(J_+J_- = J^2 - J_z^2 + \hbar J_z\)

Applying \(J^2\) to \(\vert j, m\rangle\) in general yields \(J^2\vert j, m\rangle = j(j+1)\vert j, m\rangle\)

In combining these states, we can postulate which state in the four-dimensional compound state-space will have the most negative z-component of angular momentum: intuitively, it should be \(\vert -\rangle \otimes \vert -\rangle\),

Basis for 4-D Space

(note: be consistent w/ notation for the ket vectors)

\(J^2\vert 1/2, -1/2\rangle \otimes \vert 1/2, -1/2\rangle = -1\vert 1/2, -1/2\rangle \otimes \vert 1/2, -1/2\rangle\); -1 is an eigenvalue.

\(J^2\) and \(J_z\) both acting on the state

Hence we can write \(\vert 1/2, -1/2\rangle \otimes \vert 1/2, -1/2\rangle = \vert j = 1, m = -1\rangle.\)

To this lowest state we successively apply the raising operator \(J_+\): (include full written calculations here as well) \(J_+\vert 1, -1\rangle = \vert 1, 0\rangle\) which corresponds to the state \(\dfrac{1}{\sqrt{2}}\vert +-\rangle + \vert -+\rangle\) and applying it once more gives: \(J_+\vert 1, 0\rangle = \vert 1, 1\rangle\) which is the maximal state.

Now we have three states, but our state space is four-dimensional - we are missing one basis vector, which must also be orthogonal. Since the values for \(j = 1\) are exhausted, then we must have \(j = 0\) and hence \(m = 0\) because \(m\) is bounded by \(\pm j\).

Thus, to construct the fourth state, we must take \(\vert 0, 0 \rangle\) corresponding to \(\dfrac{1}{\sqrt{2}}\vert +-\rangle - \vert -+\rangle\).

For a fixed value of \(j\), the range of values (and hence the dimension of the vector space) that \(m\) can take on are within \(\pm j\); there are \(2j + 1\) values.

In this decomposition, we have written the four-dimensional compound state space, i.e. the tensor product of two 2-dimensional spaces as the direct sum of a three-dimensional and one-dimensional space respectively. This is one case of the Clebsch-Gordan decomposition.
